# Laravel-todo
ToDo API with Laravel

# API 
## Route - /api

## Add Todo
### Route - /api/add
### Method - POST
### JSON Request(strings) - 'taskName', 'status'

## Edit Todo
### Route - /api/edit/{id}
### Method - PUT
### JSON Request(strings) - 'taskName'

## Delete Todo
### Route - /api/destroy/{id}
### Method - DELETE

## List All Todo
### Route - /api/list
### Method - GET
